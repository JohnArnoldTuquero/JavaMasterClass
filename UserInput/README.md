# User Input Practice (Scanner)

## Objective

Learn how to get input from the user using Java's `Scanner` class.

---

## Concepts to Learn

- Importing Scanner
- Creating a Scanner object
- Reading different data types
- Displaying user input
- Basic interaction with the user

---

## Methods to Research

- nextLine()
- next()
- nextInt()
- nextDouble()
- nextBoolean()
- close()

---

# Projects
