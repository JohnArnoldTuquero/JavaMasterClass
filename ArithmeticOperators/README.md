# Arithmetic Operators

Learn how Java performs basic mathematical operations using arithmetic operators.

---

# Objectives

After completing this lesson, you should be able to:

- Use arithmetic operators correctly.
- Perform calculations with numbers.
- Store calculation results in variables.
- Display results using `print`, `println`, and `printf`.
- Accept numbers from the user using `Scanner`.

---

# Concepts to Learn

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Modulus / Remainder (`%`)
- Operator Precedence
- Parentheses
- Integer Division
- Floating-point Division

---

# Operators

| Operator | Meaning | Example |
|----------|---------|---------|
| + | Addition | 10 + 5 |
| - | Subtraction | 10 - 5 |
| * | Multiplication | 10 * 5 |
| / | Division | 10 / 5 |
| % | Modulus (Remainder) | 10 % 3 |
