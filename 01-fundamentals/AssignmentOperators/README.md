# Assignment Operators

| Operator | Meaning | Example |
|----------|---------|---------|
| = | Assign value | x = 10 |
| += | Add then assign | x += 5 |
| -= | Subtract then assign | x -= 5 |
| *= | Multiply then assign | x *= 5 |
| /= | Divide then assign | x /= 5 |
| %= | Modulus then assign | x %= 5 |
